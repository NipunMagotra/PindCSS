Pind CSS

The easiest way to style your website using just HTML classes. No setup required!

Beginner Friendly v1.0.0

How to install

Just put this script tag in your HTML file and you are good to go.

<script src="engine.js"></script>

1. Spacing

Just type a number to add pixels. E.g `p-20` adds 20px padding.

  • Padding: p-20 (all sides), pt-10 (top), px-20 (left and right)
  • Margin: m-10 (all sides), mb-40 (bottom), mx-auto (center)
<div class="p-20 mt-10">Padded Box</div>
p-20 adds a lot of space inside here

2. Colors

Use curated color names for beautiful results, or raw hex codes. No hashtags needed.

  • Basic: bg-black, bg-red, bg-green, bg-lemon, bg-yellow, bg-blue
  • Theme: bg-ocean, bg-strawberry, bg-honeydew, bg-sky, bg-steel
  • Text Color: c-honeydew, c-ocean, c-white
<div class="bg-strawberry c-honeydew p-20">Strawberry Red</div>
We picked a Strawberry background!

3. Text

Make your text look exactly how you want it.

  • Size & Boldness: fs-32 (Font size 32px), fw-700 (Bold)
  • Alignment: text-center, text-left, text-right
<p class="fs-32 fw-700 text-center c-457b9d">Large Text</p>

Big Text

4. Borders and Radius

Draw lines around boxes and make the corners round easily.

  • Radius: radius-10 (Make it slightly round), radius-100p (Make it a circle)
  • Borders: border-1d3557 (Use color to add a border), border-none (Remove borders)
<div class="border-1d3557 radius-10 p-20">Rounded</div>
Round Box

5. Layout Settings

Put things exactly where you want them side by side.

  • Display: flex, grid, hidden
  • Flex Settings: col, row, center, gap-20
  • Size: w-100p (100% width)
<div class="flex row center gap-20 w-100p">...</div>
Box 1
Box 2

6. Cool Advanced Effects

Want to impress people without writing crazy CSS? Use built-in effects.

  • Frosted Glass: Just write glass to make a frosty blurry background.
  • Gradients: Write grad-sunset, grad-ocean, grad-candy, or grad-lemon to instantly apply beautiful gradients.
<div class="grad-ocean p-20 c-white">Wow!</div>
Pure glass effect!
